# SQL_Project_Music_Store_Analysis
SQL project to analyze online music store data

SQL-based data analysis project exploring a music store database to generate business insights on sales, customers, revenue, and top-performing artists.

## Database and Tools
* Postgre SQL
* PgAdmin4
